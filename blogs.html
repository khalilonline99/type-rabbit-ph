What is the difference between local and sessions storage?

Local Storage: The data stored in localStorage persists until explicitly deleted. Changes made are saved and available for all current and future visits to the site. It is used for use for long term use.
Session Storage: Changes are only available per tab. Changes made are saved and available for the current page in that tab until it is closed. Once it is closed, the stored data is deleted. It is used use when you need to store somthing that changes or somthing temporary

What is the difference between global and block scope?

Global Scope: It can be accessed beyond any block. Variables declared Globally (outside any function) have Global Scope. (var, let and const) They all have global scope
Block Scope: It is only accessed within a block. ES6 introduced two important new JavaScript keywords: let and const. These two keywords provide Block Scope in JavaScript. Variables declared inside a { } block cannot be accessed from outside the block
